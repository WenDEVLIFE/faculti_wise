# .agents Structure

This folder stores planning and implementation knowledge for Faculty_Wise.

## Existing implementation tracks
- `brainstorming/` - concept notes and feature ideation
- `ui-implementation/` - UI/UX skill documents
- `ai-implementation/` - AI scheduling skill documents
- `firebase-implementation/` - Firebase backend skill documents
- `project-structure/` - project and code organization skill documents

## Added coordination tracks
- `_templates/` - reusable templates for new skills and docs
- `decisions/` - architecture decision records (ADR)
- `roadmap/` - milestones and phased rollout plans
- `tasks/` - actionable backlog and status tracking

## Recommended workflow
1. Start in `brainstorming/` for ideas and requirements.
2. Convert approved ideas into implementation skills.
3. Record technical choices in `decisions/`.
4. Track releases in `roadmap/` and execution in `tasks/`.
